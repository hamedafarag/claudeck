    New DAG

    What is a DAG?

    A DAG (Directed Acyclic Graph) lets you run agents in parallel when they have no dependencies, and sequentially when one depends on another's output.

    1Drag agents from the palette onto the canvas
    2Connect outputs → inputs by dragging between ports to define dependencies
    3Click an edge to remove a connection
    4Agents on the same level run in parallel, dependent agents wait

    Session / Total Cost
    
          
    
            
    Every message carries a fixed ~20k token overhead from the Claude Code SDK — this includes the system prompt (~12-15k), core tool schemas (~4-5k), and environment context (~1k).
    
            
    This cost is unavoidable and cannot be reduced. Disabling optional tools in Session settings saves only ~2-3k tokens.
    
            
    Subsequent turns benefit from Anthropic's prompt caching, which significantly reduces the effective cost.
